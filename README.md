# Resume Builder

## Project Description
Resume Builder is a web application that helps users create professional resumes quickly using modern templates.

## Core Features
- Create Resume
- Edit Resume
- Live Preview
- Download Resume as PDF
- Multiple Resume Templates

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## How to Run

```bash
npm install
npm run dev
```

The application will run at:
http://localhost:3000
